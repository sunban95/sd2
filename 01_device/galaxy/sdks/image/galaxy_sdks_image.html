Overview

Image Filter SDK offers you various types of filters for processing your images and applying various effects. You can apply filters such as Sepia, Pastel, and Cartoon effects.

Figure 1: Applying various filters to an image

You can use Image Filter SDK to:

  • choose from 33 filter effects

  • adjust filter levels

  • adjust the transparency

Choosing a Filter Effect

Image Filter SDK offers you 33 filter effects. You can adjust the level of the filter effect for some of these filters.

Adjusting Filter Levels

Image Filter SDK offers you the following filter levels for filters with level adjustment:

  • Very Small

  • Small

  • Medium

  • Large

  • Very Large

Figure 2: Applying different filter levels to the sepia filter
Adjusting Transparency

You can adjust the level of transparency for an image by entering an Alpha value. The higher the Alpha value is, the less transparent the image.

Figure 3: Adjusting image transparency
Restrictions

Image Filter SDK has the following restrictions:

  • Devices with Android 4.1 Jelly Bean (API level 16) or higher support Image Filter.

  • Q01How does the gesture sensor work?
    A

    The gesture sensor transmitter emits infra-red rays that reflect off the device user’s hand. The gesture sensor receiver senses changes in these reflected infra-red rays. Air Gesture uses the data from the sensor, located to the right of the speaker on top of the device, to answer calls, select songs, and to move from one webpage to another with hand gestures.
