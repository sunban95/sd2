OverView

Sensor Extension SDK allows the user to use sensors which are not supported by Android. The SDK provides meaningful data from sensors in the Samsung Device.

Sensor Extension SDK supports the feature below.

  • Ultraviolet(UV) Index

  • Ultraviolet(UV) Index

According to further updates, the supported sensor will also be added.

UltraViolet Index

The Ultra Violet feature provides UV Index from the outdoor environment.

Figure 1: UV sensor

To measure the ultraviolet rays, the device uses the sensor located at the rear side of the phone.

The sensor needs to be placed directly under the sun for 5 seconds in order to measure the ultraviolet rays.

UV index values have 12 levels.

UV index values have 12 levels.
UV Index Rating Description
< 2 Low No danger to the average person
3 ~ 5 Moderate Little risk of harm from unprotected sun exposure
6 ~ 7 High High risk of harm from unprotected sun exposure
8 ~ 10 Very High Very high risk of harm from unprotected sun exposure
11+ Extreme Extreme risk of harm from unprotected sun exposure
Note

In natural indoor environment, UV Index is observed to be almost at level 0.

HRM Sensor IR/RED Signal

The HRM Sensor is located on the rear panel of the phone. If you place your finger, measurement starts.

The photoplethysmogram(PPG) signal means the amount back of IR / RED light that reflected from blood vessel in the finger.

Figure 2: HRM sensor

Restrictions

Sensor Extension SDK has the following restrictions:

  • Ultraviolet Index

    • Devices with Android 4.4 Kit Kat (API level 19) or higher required.
    • Only supported by Samsung device with Ultraviolet sensor.

  • Ultraviolet Index

    • Devices with Android 4.4 Kit Kat (API level 19) or higher required.
    • Only supported by Samsung device with Ultraviolet sensor.

FAQ

  • Q01How does the gesture sensor work?
    A

    The gesture sensor transmitter emits infra-red rays that reflect off the device user’s hand. The gesture sensor receiver senses changes in these reflected infra-red rays. Air Gesture uses the data from the sensor, located to the right of the speaker on top of the device, to answer calls, select songs, and to move from one webpage to another with hand gestures.
