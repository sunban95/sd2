Change History

  • Samsung Sensor Extension SDK 1.1.4 – Mar 28, 2016

    • Samsung Sensor Extension SDK 1.1.4 has been released with enhanced stability.

  • Samsung Sensor Extension SDK 1.1.3 - Nov 27, 2015

    • Modified the Android M permission issue.

  • Samsung Sensor Extension SDK 1.1.2 - Aug 17, 2015

    • Modified to load the UV library after checking the version of software platform.

  • Samsung Sensor Extension SDK 1.1.1 - Mar 2, 2015

    • Changed process for requesting a Sensor Extension SDK with your Samsung Account

  • Samsung Sensor Extension SDK 1.1.0 - Jan 14, 2015

    • The algorithm of the Sensor Extension has dependencies based on the chipset type. Support on further models using different HRM sensors.

  • Samsung Sensor Extension SDK 1.0.4 - Nov 14, 2014

    • The algorithm of the Sensor Extension has dependencies based on the chipset type. Support on further models using different UV sensors.

    • The package name has been modified. samsung.android.sdk.ssensorextension -> com.samsung.android.sdk.sensorextension

Features

  • Measuring UV Index from the sun.

  • Measuring HRM raw data from your finger including IR/RED LED raw data.
