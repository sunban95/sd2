Change History

  • Samsung Remote Sensor SDK 1.0.1

    • Support for the Andorid phones, not limited to Samsung.

      • Supported devices vary depending on your region, operator and device brand.
  • Samsung Remote Sensor SDK 1.0.0

    • Samsung Remote Sensor SDK 1.0.0 has been released with enhanced stability and better performance.

  • Samsung Remote Sensor SDK 1.0.0 beta

    • The Remote Sensor is separated from the Samsung Mobile SDK to make an efficient environment for the application development.

    • New permission will be required when you initialize Remote Sensor. Refer to programming guide for more information.

Features

  • Get user activity data from the wearable device.

  • Get pedometer data from the wearable device.

  • Check whether the user is wearing the wearable device.
