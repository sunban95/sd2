Samsung In-App Purchase 2 ANE Library

Samsung In-App Purchase (hereinafter “IAP”) version 2 provides an IAP Air Native Extension (hereinafter “ANE”) library which is supported in Adobe Air. The ANE library is developed based on the IAP library for Android.

Note

For more information about IAP 2 and the IAP 2 library, download the following guide from the IAP section of the Samsung Developers site.

The 3rd Party Adobe Air Mobile Application communicates with the IAP Android package through ANE as shown in the following image. The IAP library for Android and the Air library, which transmission makes interlocking possible, are both included in the ANE library. The 3 rd Party Air Mobile Application actually interlocks with the IAP 2 library for Android through this Air library.
