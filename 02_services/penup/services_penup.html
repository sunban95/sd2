Overview

PEN.UP is a creative social network service for digital arts. Users can share their own artworks as the artist, follow other artists, collect artworks they like, and communicate with other users using comments and the fanbook. If you want to find out more about the PEN.UP service, please visit PEN.UP’s Website.

The PEN.UP SDK helps to integrate PEN.UP features into your application. The deep linking feature, provided when an artwork is posted using the PEN.UP SDK, can be a great opportunity to promote your drawing application. This will help many PEN.UP users, who are not familiar with your drawing application, to easily discover and install it.

What is PEN.UP SDK?

The PEN.UP SDK enables the following features:

  1. Authentication

    • Connecting to the PEN.UP service based on OAuth 2.0

    • Supporting Single Sign-On (SSO) with the PEN.UP application whose version is 1.5.0 or above

  2. Post PEN.UP resources

    • Posting a new artwork or adding a new collection

    • Post Dialog which provides a simple interface to post an artwork easily

    • Deep linking to your application when posting an artwork

  3. Access to PEN.UP resources

    • Getting information such as artists, artworks, collections, and tags

  • Test
    Your App
    You should test your app with debug client ID first.
  • Upload Your App
    to Google Play
    You can upload your app to Google Play. (The deep link is supported only for Google Play.)
Note

If your application can post an artwork to PEN.UP using PEN.UP SDK, please email us at penup.sdk@samsung.com after uploading your application to Google Play. Then, after the verification process, we’ll expose your application more in PEN.UP.

Restrictions

  • Android SDK version: Android 4.0 Ice Cream Sandwich (API level 14) or above

  • Supported devices : Android 4.0 ICS devices or above

FAQ

  • Q01Why is my drawing application not displayed in “Apps for PEN.UP” even though I’ve implemented to post an artwork into PEN.UP using the PEN.UP SDK and uploaded the application to Google Play?
    A

    Please email us at penup.sdk@samsung.com after uploading your application to Google Play. After the verification process, your application will be displayed in Apps for PEN.UP.

  • Q02What features can I implement with the PEN.UP SDK?
    A

    Post an artwork into PEN.UP and access PEN.UP resources using PEN.UP SDK. Especially, if you’re developing a drawing application, we strongly recommend you to implement to post an artwork into PEN.UP for the deep linking.

  • Q03What should I do before implementing PEN.UP features?
    A

    You should first register your application here. After that, we’ll issue the client ID for authentication and send it to you. You can implement PEN.UP features with the client ID.
