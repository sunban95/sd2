FAQ

  • Q01Why is my drawing application not displayed in “Apps for PEN.UP” even though I’ve implemented to post an artwork into PEN.UP using the PEN.UP SDK and uploaded the application to Google Play?
    A

    Please email us at penup.sdk@samsung.com after uploading your application to Google Play. After the verification process, your application will be displayed in Apps for PEN.UP.

  • Q02What features can I implement with the PEN.UP SDK?
    A

    Post an artwork into PEN.UP and access PEN.UP resources using PEN.UP SDK. Especially, if you’re developing a drawing application, we strongly recommend you to implement to post an artwork into PEN.UP for the deep linking.

  • Q03What should I do before implementing PEN.UP features?
    A

    You should first register your application here. After that, we’ll issue the client ID for authentication and send it to you. You can implement PEN.UP features with the client ID.
